Linux下为程序生成快捷方式

Contents

  • 前言
  • 配置

前言

下载程序后每次执行都要运行对应的 .sh 文件, 这样很麻烦, 所以要生成快捷方式.

环境: ubuntu 18.04

配置

在 /usr/share/applications/ 目录下新建一个 .desktop 文件:

sudo vim /usr/share/applications/wechat_web_devtools.desktop

然后配置如下:

[Desktop Entry]

Type=Application

Name=微信开发者工具

GenericName=微信开发者工具

Comment=linux版本的微信开发者工具

Exec=sh /home/jr/Downloads/wechat_web_devtools/bin/wxdt

Icon=/home/jr/wechat_web_devtools/dist/package.nw/static/image/logo@2x.png

Terminal=0

其中 Exec 是对应的执行路径, Icon 是图标的路径.

配置完执行在 /usr/share/application 目录下双击刚才的文件就可以运行程序了.
