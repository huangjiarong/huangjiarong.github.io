解决从Github上git clone速度慢的问题

Contents

  • 问题
  • 解决

问题

今天从github上下载个微信开发者工具, 奈何速度20k/s, 网上查了下许多都是说改hosts的, 但是我改了hosts速度还是那样, 看到一篇博文说可以用码云, 果然速度比之前的快了很多.

解决

  1. 从github上把项目fork到自己的仓库, 一下子解决.
  2. 码云建仓库时可以从github上导入自己的仓库, 导入可能需要一点点时间.
  3. 执行git clone 码云上的项目地址 --depth=1, 速度快得飞起.
